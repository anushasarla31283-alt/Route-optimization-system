# Route Optimization System

## Project Description
This project finds the shortest route between cities using Dijkstra's Algorithm.

## Technologies Used
- Java
- Data Structures and Algorithms
- Graphs
- Dijkstra Algorithm

## Features
- Find shortest path between cities
- User input for source and destination
- Distance calculation
- Route display
- City validation

## Cities Included
- Hyderabad
- Warangal
- Vijayawada
- Visakhapatnam
- Tirupati
- Rajahmundry
- Kurnool
- Anantapur
- Nellore
- Khammam

## Sample Output

Enter Source City: Hyderabad

Enter Destination City: Tirupati
